# sample_mobile
用vue.js製作的手機demo, 配色及版面模仿網易雲音樂App, 圖片來自於網路上的動物圖片.
* 目前僅實作 "我的"、"發現"、"朋友"、"視頻"展示頁面, 其餘功能尚未實作.
* 此專案使用yarn管理


## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
